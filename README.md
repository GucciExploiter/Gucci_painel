<h2 align="center">💻  Features 💻</h2>

# Gucci_painel
Um painel basico de consultas ele esta em beta e n tem quase nada

- [x] Consulta de CEP
- [x] Consulta de CNPJ
- [ ] Consulta de CPF
- [x] Consulta de BIN
- [ ] Consulta de Nome
- [ ] Consulta de Número
- [ ] Consulta de Email
- [ ] Consulta de Placa
- [x] Geolocalizador de IP
- [ ] Consulta RG
